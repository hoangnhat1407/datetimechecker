# Topic 1 - Unit Testing

Double-click `run.bat` to run the unit test suite.

The runner:

- runs Maven tests from this folder;
- uses test source files from `Topic 1/src/test/java`;
- uses `Topic 1/test-data.json` for parameterized date validation cases;
- prints the JUnit console output and a Surefire summary table;
- saves each run under `Topic 1/reports/run-YYYYMMDD-HHMMSS`.

Main application source is still compiled from `../src/main/java`.
